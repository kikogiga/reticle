Benvingut a la Reticle!

Un marc super simple, sensible construït per funcionar en dispositius mòbils, tabletes, portatils i ordinadors d'escriptori.

Reticle es desenvolupa en LESS, un poderós CSS pre-processador que ajuda a escriure més net i organitzat, CSS, ben estructurat que es pot mantenir fàcilment amb el temps. Base suporta tots els principals navegadors i navegadors antics com Internet Explorer 7.

Introducció

Abans de començar, vostè ha de tenir els següents requisits:

  1. Un bon coneixement d'HTML i CSS
  2. La versió més recent de Reticle - Descàrrega desde Github
  3. Un editor de codi impressionant com Sublime Text 2
  4. Un compilador LESS
    • Prepros per a Windows
    • CodeKit per a Mac
  5. Calma i una ment oberta :)

Què ve amb Reticle?

Una descàrrega petita i minimalista que inclou el següent:

  • index.html - Plantilla d'HTML bàsic que inclou jQuery
  • style.css - Estils de tipografia, encapçalats, taules, elements de formulari, etc
  • readme.markdown - El fitxer Llegiu-me de Base (no dubti a suprimir aquest)
  • less (carpeta):
    • mixins.less - Afegits bàsics que inclouen les propietats font-size i més
    • style.less - El full d'estil de base en format LESS
  • images (carpeta):
    • favicon.ico - Estàndard 32x32 favicon
    • apple-touch-icon (x3) - Els tres mides d'icones tàctils d'Apple per l'iPhone 3, iPad i iPhone 4 amb la retina
  • js (carpeta):
    • plugins.js - Arxiu javascript per defecte (inclou connectors i scripts combinats)
    • scripts.js - Arxiu javascript per defecte (inclou connectors i scripts combinats)
