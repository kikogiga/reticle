Un marc super simple, sensible construït per funcionar en dispositius mòbils, tabletes, portatils i ordinadors d'escriptori.

Reticle es desenvolupa en LESS, un poderós CSS pre-processador que ajuda a escriure més net i organitzat, CSS, ben estructurat que es pot mantenir fàcilment amb el temps. Base suporta tots els principals navegadors i navegadors antics com Internet Explorer 7.

Busseig en l'estructura de Reticle

Inclou el següent conjunt d'arxius:
  • Arxius de LESS per personalitzar fàcilment el seu full d'estil de Reticle
  • Bàsic de la plantilla HTML5 que inclou jQuery per començar
  • Arxiu javascript per defecte amb millores de base i modernizr.js i fittext.js
  • Les tres mides d'icones tàctils d'Apple per l'iPhone 3, iPad i iPhone 4 amb retina + el favicon estàndard per a tots els navegadors web
  • Base d'estils que conté estils de tipografia, comentaris, cites, taules, formularis i més + una reixeta 12 columnes flexibles
